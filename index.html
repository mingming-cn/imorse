IMorse

Project imorse is a pure JavaScript (1.5 kb) library for encoding/decoding morse code messages, unicode supported.

项目 imorse 是一个 JavaScript 开发的摩斯密码编码库(1.5 kb)，可以支持 unicode，例如中文摩斯密码。


Space: Short: Long: 



  
 Install 

  
npm install imorse 

  
 Then import it. 

  
// import library use script tag.
<script type="text/javascript" src="dist/imorse.min.js"></script>

// or ES6 style.
var WordWidth = require('imorse');


  
Usage

  
imorse.encode('Hello, IMorse!');
// unicode
imorse.encode('コンニチハ, セカイ!');
imorse.encode('越过长城，走向世界');

imorse.decode('../.-../---');
